## Usage

```
static-server [--port <PORT>] [--root <DIRECTORY>]
```

Arguments (all are optional):

* `--port`: [`Number`] port number, defaults to 3000
* `--root`: [`String`] root folder, defaults to working directory

For example:

```bash
$ static-server --port 8899 --root ./
```
